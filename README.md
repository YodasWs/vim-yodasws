# vim-yodasws
Making Vim so Much Easier!

## Vim 8 Installation

1. Go to `vimfiles` folder
	```bash
	# Linux
	cd ~/.vim/
	# Windows 10
	cd /mnt/c/Users/[username]/vimfiles/
	```

1. Clone git repo

	```bash
	mkdir -pv pack/github/start
	cd pack/github/start

	git clone git@github.com:YodasWs/vim-yodasws.git
	```
